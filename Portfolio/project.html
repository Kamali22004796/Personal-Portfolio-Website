Smart Wastewater pH Monitoring and Alert System

Household wastewater is collected into a tank where chemicals are automatically added to balance the pH level between 6 and 7.5. A pH sensor placed in the drainage monitors the disposed water in real time. If the pH goes below 6 or above 7.5, an instant alert is sent to the house owner via the Blynk mobile app. This prevents water pollution in nearby water bodies and the treated water can be reused for plants, toilets, and other household purposes. Tech Stack: ESP32 · pH Sensors · Blynk App · Python · IoT

Bitcoin Price Prediction Web Application

Many people invest in Bitcoin without proper knowledge of price trends. This web application allows users to enter any date — past or future — and get an approximate predicted Bitcoin price. Built using machine learning models including Linear Regression, Random Forest, and LSTM trained on historical Bitcoin data, with an interactive Streamlit interface for easy use. Tech Stack: Python · Scikit-learn · Pandas · LSTM · Streamlit
